## :information_source: How To Use

---

```bash
# Create a project using this boilerplate
$ yarn create next-app -e https://github.com/eliziebluiz/NLW04.git

# ✔ What is your project named?
$ Move it

# Install dependencies
$ yarn install

# Run the app
$ yarn dev

# Run tests
$ yarn test:watch

# Run build
$ yarn build
```
